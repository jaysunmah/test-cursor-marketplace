# Compliance Readiness Studio

Compliance Readiness workflows for compliance preparation, control reviews, and evidence collection packaged as a reusable studio-style plugin.

## Included Components

- 2 skills for recurring workflows
- 2 rules for persistent guidance
- 2 specialized agents for focused tasks
- 1 slash command for quick entry points

## Structure

```
compliance-readiness-studio/
├── .cursor-plugin/
│   └── plugin.json
├── agents/
├── commands/
├── rules/
├── skills/
└── README.md
```

## Themes

This plugin is aimed at security work and is intentionally packaged with multiple realistic, focused components rather than a single demo file.
