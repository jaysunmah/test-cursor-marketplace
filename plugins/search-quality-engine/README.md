# Search Quality Engine

Search Quality workflows for search relevance, query behavior, and result quality review packaged as a reusable engine-style plugin.

## Included Components

- 2 skills for recurring workflows
- 2 rules for persistent guidance
- 2 specialized agents for focused tasks
- 1 slash command for quick entry points

## Structure

```
search-quality-engine/
├── .cursor-plugin/
│   └── plugin.json
├── agents/
├── commands/
├── rules/
├── skills/
└── README.md
```

## Themes

This plugin is aimed at development work and is intentionally packaged with multiple realistic, focused components rather than a single demo file.
