# Accessibility Review Kit

Accessibility Review workflows for accessible product design and interaction quality packaged as a reusable kit-style plugin.

## Included Components

- 2 skills for recurring workflows
- 2 rules for persistent guidance
- 2 specialized agents for focused tasks
- 1 slash command for quick entry points

## Structure

```
accessibility-review-kit/
├── .cursor-plugin/
│   └── plugin.json
├── agents/
├── commands/
├── rules/
├── skills/
└── README.md
```

## Themes

This plugin is aimed at design work and is intentionally packaged with multiple realistic, focused components rather than a single demo file.
